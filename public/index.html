Tasks API

Account

POST API/register : Create your account

POST API/login : Login to your account to get your JWT Token

{
'username': 'USERNAME',
'password': 'PASSWORD'
}

POST API/test : Test your token

{
'Authorization': 'Bearer xxxxxxxxx.xxxxxxxxxx.xxxxxxx'
}


GET

GET API/task-lists : To get all the task lists

GET API/task-lists/:id : To get a specific task list

GET API/task-lists/:idTaskList/:idTask : To get a task in a specific task list


POST

POST API/task-list : Add a new task list

{
'title': 'NEW TASK LIST',
'tasks': '[]'
}

POST API/task-lists/:id : Add a new task in a specific task list

{
'title': 'NEW TASK',
'description': 'DESCRIPTION',
'priority': 'PRIORITY',
'startDate': 'STARTDATE',
'endDate': 'ENDDATE',
'advancement': 'ADVANCEMENT'
}


PUT

PUT API/task-lists/:id : Update a specific task list

{
'title': 'UPDATED TASK LIST',
'tasks': '[]'
}

PUT API/task-lists/:idTaskList/:idTask : Update a task in a specific task list

{
'title': 'UPDATED TASK',
'description': 'DESCRIPTION',
'priority': 'PRIORITY',
'startDate': 'STARTDATE',
'endDate': 'ENDDATE',
'advancement': 'ADVANCEMENT'
}


DELETE

DELETE API/task-lists/:id : Delete a specific task list

DELETE API/task-lists/:idTaskList/:idTask : Delete a task in a specific task list
